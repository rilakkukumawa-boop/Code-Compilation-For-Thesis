# Code-Compilation-For-Thesis
This code possesses all code that was done on the Thesis for Supernova Remnants and HII Regions. Currently work in progress at the moment.
Please press on the button labelled as 'main' above this window to access all working directories of code that was used in current thesis
for Supernova Remnants and HII Regions of M33. 

/BRIEF DESCRIPTIONS:
- **Alignment-Code**
This code is used to align images and apply WCS where necessary. 
- **Calibration-Code:**
This code is used to calibrate raw science images. 
- **Continuum-Subtraction:**
This code is used to preform extra background subtraction, and preform star detection with aligned/cropped cutouts via DS9 or Python. 
- **Evolutionary-Parameter-Codes:**
This code is used to calculate the diameter of aperture regions
- **Ratio-Codes:**
An overview of the Ratio Map demos, and also ratio maps that were used. 

Please ignore when messages say 'error 404', CONTINUE to load files from left hand side of screen by clicking the suggested code.
